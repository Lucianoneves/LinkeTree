# LinkeTree

Uma página estilo “link-in-bio” para centralizar links (redes sociais, portfólio, contato) em um único lugar.

## Visão Geral

O LinkeTree é um projeto front-end construído com React + TypeScript e Vite, pensado para ser simples de personalizar e fácil de publicar.

## Funcionalidades

- Página única com lista de links
- Layout responsivo (mobile e desktop)
- Personalização rápida de textos e links
- Build otimizado para deploy

## Tecnologias

- React
- TypeScript
- Vite

## Como Rodar Localmente

1) Instale as dependências:

```sh
npm install
```

2) Inicie o servidor de desenvolvimento:

```sh
npm run dev
```

3) Acesse o endereço exibido no terminal (normalmente):

- http://localhost:5173/

## Scripts

```sh
npm run dev
npm run build
npm run preview
```

## Deploy

Gere a build e publique a pasta `dist` em qualquer serviço de hospedagem estática.

```sh
npm run build
```

## Personalização

Atualize os dados (nome, bio e links) nos componentes do projeto para refletir suas informações.
